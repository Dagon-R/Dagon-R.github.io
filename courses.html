Computer Science Courses

Introductory classes covering the same subjects and skills as entry level courses in a standard university curriculum.

These courses are designed with true beginners in mind, whether a younger student or a curious professional looking to learn a new skill or hobby. Most of the more advanced classes require that "Introduction to Computer Science" be taken first, or that the concepts listed under that class description are understood beforehand.


Classes are twice weekly, with the first class being taught once each Monday through Wednesday, and the second on Thursday through Saturday to allow for more flexible attendence. Students will need Administrator access to their computers (i.e, the ability to download, install, and run arbitrary programs).

Embedded software is any kind of code that runs on a computer that isn't a PC. This ranges from consumer electronics (such as bluetooth speakers and smart TVs) all the way up to critical control systems (such as in avionics and self driving cars). This class uses simulated hardware for practice, but a beginners hardware kit can be included for an extra fee.

It is recommended that the Introductory course is taken first

Working in Java and Android studios, learn how to build a simple application that runs on an Android device (or emulator). The cumulative goal of this class is to finish with a simple but functional app that could hypothetically be published on Google Play.

It is recommended that the Introductory course is taken first

Working with HTML, CSS, Javascipt, Git, and the Bootstrap library, learn to create simple but effective web pages (like this one!). Although the artisitic choices fall more under graphic design than computer science, the skills here will be useful to anyone who wants to work with websites in the future. At the end of the class, use Github's free hosting to upload your site and make it publicly visible.

It is recommended that the Introductory course is taken first

If you've ever wanted to learn more about the actual hardware of your computer, this is a great class to get familiar with the terminology. By the end of this class, you'll be able to talk about a computer's specifications (such as clock speeds, cache sizes, and core count), as well as make a list of components to actually build your own computer.
