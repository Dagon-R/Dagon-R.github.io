Computer Science Courses

Introductory classes covering the same subjects and skills as entry level courses in a standard university curriculum.

These courses are designed with true beginners in mind, whether a younger student or a curious professional looking to learn a new skill or hobby. Most of the more advanced classes require that "Introduction to Computer Science" be taken first, or that the concepts listed under that class description are understood beforehand.


Classes are once weekly which each classes' schedule being designed to fit the needs of the particular students who request the class. Students will need Administrator access to their computers (i.e, the ability to download, install, and run arbitrary programs).

For a younger student looking to get started with coding and engineering, this class is built off of the LEGO Spike System.
Learn to understand basic concepts about code, including variables, loops, and control statements. Also learn about prototyping, iterative testing, and mechanical engineering!
The course covers at least 8 classes, and includes a LEGO Spike kit for the student to take home. For families who have multiple interested children kits can be shared to reduce class cost.
Course fee: $600

Embedded software is any kind of code that runs on a computer that isn't a PC. This ranges from consumer electronics (such as bluetooth speakers and smart TVs) all the way up to critical control systems (such as in avionics and self driving cars).
This class includes a beginners hardware kit for each student, including an Arduino Uno board, as well as plethora of components ranging from basic resistors and jumper wires, all the way up to infrared controllers. Also included is a series of extra project documentation outside the scope of the class, designed by Elegoo.
Course fee: $400

If you've ever wanted to learn more about the actual hardware of your computer, this is a great class to get familiar with the terminology. By the end of this class, you'll be able to talk about a computer's specifications (such as clock speeds, cache sizes, and core count), as well as make a list of components to actually build your own computer.
